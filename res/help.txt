=== General

Ctrl + Q   exit immediatly
Ctrl + R   reload shaders
Ctrl + D   toggle debug menu
Ctrl + F   toggle fullscreen
`          toggle console

=== Console

Insert     replace string with one from clipboard
Ctrl + V   put string from clipboard
Ctrl + C   copy input to clipboard
Ctrl + U   clear input string after cursor
Down       clear whole input string
Left\Right move input cursor
Home       move cursor to the beginning
Page Up    go to older log page
Page Down  go to newer log page
End        go to current page
Ctrl + <=  shift output to left
Ctrl + =>  shift output to right
Ctrl + ^   reset output shift

=== Debug menu

1-9       switch tab
a-z       select section / toggle option in selected section
Escape    deselect section
Tab       disable input capture
Ctrl + D  enable input capture / exit menu
